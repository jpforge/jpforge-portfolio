## 🛠️ Setup Instructions

1. Clone the repo:
```bash
git clone https://github.com/your-username/jpforge-portfolio.git
cd jpforge-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
